# 拓词桌面客户端（非官方）
![Logo](/toWords/css/images/logo.png)

## 简介
[拓词](http://www.towords.com/)是一款在线英语学习产品，本软件在基于网络版的数据的同时重新设计了用户界面，方便拓词用户使用。

## 安装

直接单击本页面右侧的**Download ZIP**下载软件压缩包，解压后运行toWords.exe即可。
